# JobHunt OS

> An AI-native job hunt operating system for anyone running a serious job search.

**Live:** https://shreyamittal007.github.io/jobhunt-os/

This repo hosts the public landing page and waitlist. JobHunt OS is an AI-native operating system that tailors your resume per role, drafts personalised outreach to hiring managers and recruiters, tracks your pipeline, and preps you for interviews — all in one agentic pipeline.

The product itself (CLI + automation) is private. [Join the waitlist](https://shreyamittal007.github.io/jobhunt-os/#waitlist) for early access.
